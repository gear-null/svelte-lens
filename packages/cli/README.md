# @gear-null/svelte-lens-cli

CLI to add [svelte-lens](https://npmjs.com/package/svelte-lens) to your Svelte project.

## Usage

```bash
npx svelte-lens init
# or
npx sv-lens init
```

Detects SvelteKit and Vite+Svelte projects, and wires svelte-lens into your `+layout.svelte` automatically.
